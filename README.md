# Jupiter Academy Content

Community-maintained lessons and glossary for [Jupiter Academy](https://academy.jup.ag).

## Structure

```
lessons/          MDX lesson files
glossary/
  terms.json      Glossary terms and definitions
```

## Contributing

We welcome contributions! See [CONTRIBUTING.md](CONTRIBUTING.md) for guidelines.

- **Fix a typo or improve a lesson** - Edit the MDX file and open a PR
- **Add a new lesson** - Create a new `.mdx` file in `lessons/` following the frontmatter format
- **Add a glossary term** - Add an entry to `glossary/terms.json`
